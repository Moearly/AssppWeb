### 5. 智能账号选择器 ✅

**文件**: `backend/src/services/accountSelector.ts`

实现功能：
- ✅ 按地区选择账号
- ✅ 负载均衡（使用次数最少）
- ✅ 冷却机制（避免频繁使用）
- ✅ 健康状态过滤（只选active）
- ✅ 频率限制检查
- ✅ 统计信息

选择策略：
1. 筛选：地区匹配 + 状态正常
2. 排序：使用次数少 → 最久未使用
3. 冷却：检查最后使用时间
4. 回退：全部冷却时选最久未用

### 6. 健康检查服务 ✅

**文件**: `backend/src/services/healthCheck.ts`

实现功能：
- ✅ 单个账号健康检查
- ✅ 批量检查所有账号
- ✅ 健康日志记录
- ✅ 健康历史查询
- ✅ 错误分类（token_expired, locked, error）
- ✅ 自动更新账号状态

健康判断：
- `healthy` - 认证成功
- `token_expired` - Token过期（2034/2042）
- `locked` - 账号被锁
- `error` - 其他错误

## 📝 使用示例

### 1. 初始化数据库

```typescript
import { initDatabase } from './db/index.js';

// 启动时自动初始化
initDatabase();
```

### 2. 添加账号到池

```typescript
import { AccountPoolService } from './services/accountPool.js';

const service = new AccountPoolService();

const account = service.addAccount({
  email: 'user@example.com',
  password: 'password123',
  country: 'US',
  deviceIdentifier: 'a1b2c3d4e5f6',
});

console.log('Account added:', account.id);
```

### 3. 智能选择账号

```typescript
import { AccountSelector } from './services/accountSelector.js';

const selector = new AccountSelector();

const accountId = await selector.selectAccount('US');
console.log('Selected account:', accountId);
```

### 4. 使用账号下载应用

```typescript
import { AccountPoolService } from './services/accountPool.js';
import { getDownloadInfo } from './services/appleClient.js';

const poolService = new AccountPoolService();
const account = poolService.getFullAccount(accountId);

const downloadInfo = await getDownloadInfo({
  appId: 123456789,
  deviceId: account.deviceIdentifier,
  passwordToken: account.credentials.passwordToken!,
  DSID: account.credentials.DSID!,
  cookies: Object.entries(account.credentials.cookies || {}).map(([name, value]) => ({
    name,
    value,
  })),
  pod: account.pod || undefined,
});

console.log('Download URL:', downloadInfo.downloadURL);
```
