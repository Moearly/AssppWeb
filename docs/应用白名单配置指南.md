## 🔧 配置方法

### 方式一：使用 curl 直接添加（推荐脚本化）

#### 添加单个应用

```bash
# 从 iTunes Search API 获取应用信息并添加到白名单
APP_ID=1508712998  # 替换为你的应用 ID
API_KEY="YOUR_ADMIN_API_KEY"  # 替换为你的管理员 API Key

curl -s "https://itunes.apple.com/lookup?id=${APP_ID}&country=us&entity=software" | \
  jq -r '.results[0] | {
    softwareId: .trackId,
    bundleId: .bundleId,
    name: .trackName,
    country: "US",
    artworkUrl: .artworkUrl512,
    version: .version
  }' | \
  curl -s -X POST \
    -H "x-admin-key: ${API_KEY}" \
    -H "Content-Type: application/json" \
    -d @- \
    http://localhost:8080/api/admin/whitelist/apps
```

#### 批量添加应用

```bash
#!/bin/bash

API_KEY="YOUR_ADMIN_API_KEY"
SERVER="http://localhost:8080"  # 或 https://chatvpn.site

# 应用 ID 列表
APP_IDS=(
  1508712998  # Spectre VPN
  932747118   # Shadowrocket
)

for APP_ID in "${APP_IDS[@]}"; do
  echo "📱 添加应用 ID: $APP_ID"
  
  curl -s "https://itunes.apple.com/lookup?id=${APP_ID}&country=us&entity=software" | \
    jq -r '.results[0] | {
      softwareId: .trackId,
      bundleId: .bundleId,
      name: .trackName,
      country: "US",
      artworkUrl: .artworkUrl512,
      version: .version
    }' | \
    curl -s -X POST \
      -H "x-admin-key: ${API_KEY}" \
      -H "Content-Type: application/json" \
      -d @- \
      ${SERVER}/api/admin/whitelist/apps
  
  echo ""
done

echo "✅ 所有应用已添加！"
```

### 方式二：使用 Web 界面

1. 访问管理后台: `http://localhost:8080` 或 `https://chatvpn.site`
2. 点击 **"管理"** → **"白名单管理"**
3. 点击 **"搜索应用"**
4. 输入应用名称或 Bundle ID
5. 从搜索结果中选择应用
6. 点击 **"添加到白名单"**

## 📋 查看白名单

### 使用 curl

```bash
# 查看所有应用
curl -s -H "x-admin-key: YOUR_API_KEY" \
  http://localhost:8080/api/admin/whitelist/apps | jq .

# 按国家筛选
curl -s -H "x-admin-key: YOUR_API_KEY" \
  "http://localhost:8080/api/admin/whitelist/apps?country=US" | jq .

# 只看启用的应用
curl -s -H "x-admin-key: YOUR_API_KEY" \
  "http://localhost:8080/api/admin/whitelist/apps?enabled=true" | jq .
```

### 使用 CLI

```bash
bash admin-cli.sh whitelist:list
```

## 🔄 更新和删除

### 更新应用信息

```bash
APP_ID=1  # 白名单中的应用 ID（不是 App Store ID）
API_KEY="YOUR_API_KEY"

curl -X PUT \
  -H "x-admin-key: ${API_KEY}" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Spectre VPN Pro",
    "version": "1.2.0",
    "enabled": true
  }' \
  http://localhost:8080/api/admin/whitelist/apps/${APP_ID}
```

### 删除应用

```bash
APP_ID=1
API_KEY="YOUR_API_KEY"

curl -X DELETE \
  -H "x-admin-key: ${API_KEY}" \
  http://localhost:8080/api/admin/whitelist/apps/${APP_ID}
```

## ⚠️ 重要提示

### 关于账号"获取"要求

添加应用到白名单后，**还需要确保账号池中的共享账号已在 App Store "获取"过这些应用**：

1. 在真实 iPhone 上登录共享 Apple ID
2. 打开 App Store
3. 搜索应用（如 Shadowrocket）
4. 点击 **"获取"** 按钮（即使是付费应用）
5. 完成"获取"流程

只有这样，用户才能通过账号池模式下载该应用。详见：[03-NoAccount错误排查与解决方案](./03-NoAccount错误排查与解决方案.md)
