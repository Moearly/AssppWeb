### 错误 2: req.query 类型错误
位置: `backend/src/routes/admin/*.ts`, `backend/src/routes/user/apps.ts`

问题: `req.query.xxx` 类型是 `string | string[]`，但函数期望 `string`

修复方案:
```typescript
// 修改前
const { status } = req.query;

// 修改后
const status = typeof req.query.status === 'string' ? req.query.status : undefined;
```
